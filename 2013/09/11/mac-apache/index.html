mac apache

Sep 11, 2013 in develop

经常忘记命令，还是自己稍微记录一下好，免得每次都去Google。

打开 apache

sudo apachectl start

關閉 Apache：

sudo apachectl stop

重開 Apache：

sudo apachectl restart
