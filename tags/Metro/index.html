安全玩你的Win8

Mar 27, 2012 in software

不错，我再三就结下才决定玩一下Windows 8，不过也是因为最近稍微闲了一点！早就体验过的不要说我火星就好了。

首先我要说，对于现阶段的Win8，千万不要使用任何的优化软件！

因为对于那些优化软件来说，就像拿着别人设定好的Ghost来装在自己的电脑上一样，而且因为很多设定和服务有关，恢复的时候还不太好恢复！才出来的系统，谁又比谁懂多少？
