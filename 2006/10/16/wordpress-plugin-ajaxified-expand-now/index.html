WordPress plugin – Ajaxified Expand NOW

Oct 16, 2006 in web

功能介紹:
利用 AJAX 技術，在不用重新整理網頁的狀態下，提供直接讀取“深入閱讀”與迴響的功能。

這個版本可說是 Ajaxified Expand Post NOW 的延伸，如果不需要顯示評論的功能，請用前述的版本就可以了。

特色
  • 語法完全相容於 XHTML 1.0 Strict。
  • 安裝容易，深入閱讀功能安裝後馬上可用；顯示迴響功能只需在佈景主題的 index.php 增加兩段程式碼即可。
  • 幾乎適用於所有佈景主題。
  • 不會在資料庫中留下任何設定資料
  • 提供數個參數，可以自訂迴響輸出的樣式。
  • 可選擇是否倒轉迴響排序，讓最新的迴響顯示在最上面。
功能展示

本站上的每篇文章幾乎都有，點選文章中的“深入閱讀 »”與“顯示迴響”即可看到效果。

安裝
  1. 解壓縮 aen.zip
  2. 將 ajaxified-expand-now.php 複製到 plugin 資料夾中（預設為 wp-content/plugins）
  3. 在後台啟用這個 plugin
參數設定

以下是一些可供設定的自訂參數：

  1. $pluginLocate - 在極少數的情況下，WordPress 自動取得的網址，與實際使用的網址並不一樣。由於安全性的考量，此時必須手動設定 plugin 的路徑，否則 XMLHTTPRequest 無法正常工作。若需要設定此參數，請輸入本 plugin 的確實位址，例如 http://yourdomain/wordpress/wp-content/plugins/ajaxified-expand-now.php，其中 yourdomain 表示你的網域名稱。設定必須要與實際使用的網域名稱完全符合，並且移除相關的註解符號，否則設定無效。
  2. $token - 代表“深入閱讀”的代碼，通常是 。
  3. $bottomLink - 是否要在展開的內容下顯示關閉連結。
  4. $loadingMsg - “讀取中”的顯示訊息
  5. $postLess - “摘要模式”連結的顯示文字
  6. $postMore - “深入閱讀”連結的顯示文字
  7. $postLinkPrefix - “深入閱讀”連結的前置字串
  8. $postLinkPostfix - “深入閱讀”連結的後置字串
  9. $commentSortOrder - 迴響顯示的排序方式：1 為預設，舊迴響在前，新迴響在後；2 為前者的相反。
  10. $commentLess - “隱藏迴響”連結的顯示文字
  11. $commentMore - “顯示迴響”連結的顯示文字
  12. $commentLinkPrefix - “顯示迴響”連結的前置字串
  13. $commentLinkPostfix - “顯示迴響”連結的前置字串
  14. $commentFormat - 迴響輸出的樣式，有以下參數可用：
    • %comment_type% - 迴響種類：有 comment 與 trackback 兩種
    • %comment_date% - 迴響日期：根據系統設定的日期格式
    • %comment_time% - 迴響時間：根據系統設定的時間格式
    • %comment_author% - 作者名稱：如果作者有填寫網站，會自動加上超連結。
    • %comment_content% - 迴響內容

如果不會修改這些設定，使用預設值通常都可以正常工作。
