最近一直在学习NodeJS，本地上玩的差不多了，总要去架设个服务器跑一下，选择了digitalocean加的$5/mo 服务，安装了个Debian，至于为什么是Debian，好吧，因为。。。

至于怎么注册DigitalOcean这里就不在详述了，从SSH登录开始吧。

SSH登录

未免麻烦，最好是选择SSH登录，官方有详细的介绍：
How To Use SSH Keys with DigitalOcean Droplets

这里要说一下，DigitalOcean每次登录的时候都会告诉我密码过期，害得我重置了无数次密码。如果你也遇到这种问题，那么就先选择Conole Access, 然后在弹出的窗口控制台进行操作，修改root密码后再在本地操作。
